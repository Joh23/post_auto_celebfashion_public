Streamlit Cloud에서 Python 버전 설정 방법 (runtime 파일 사용)
GitHub 프로젝트에 .streamlit 폴더 생성:

Streamlit 앱이 배포되는 GitHub 프로젝트의 루트 디렉토리에 .streamlit이라는 폴더를 만듭니다.
runtime.txt 파일 생성:

.streamlit 폴더에 runtime.txt라는 파일을 생성하고, 다음 내용을 추가하세요:
코드 복사
3.9
이 파일은 Streamlit Cloud에 Python 3.9 버전을 사용하도록 지정합니다.
GitHub에 푸시:

변경된 내용을 GitHub에 커밋하고 푸시합니다:
bash
코드 복사
git add .streamlit/runtime.txt
git commit -m "Set Python version to 3.9"
git push origin main
Streamlit Cloud에서 앱 재배포:

Streamlit Cloud에서 앱이 자동으로 재배포되며, Python 3.9 버전을 사용하게 됩니다.
